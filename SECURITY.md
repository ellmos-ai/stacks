# Security Policy

`stacks` is a documentation and schema-reference repository. It contains no
executable application code, no credentials, and performs no network calls itself.

## Scope

Security issues in the modules referenced by a stack's manifest (e.g.
`ellmos-ai/ellmos-stack`, `ellmos-ai/agent-ops-stack`) should be reported to those
repositories directly, not here.

## Reporting a Vulnerability

If you find an issue specific to this repository (e.g. a broken/malicious link in the
catalog, or an error in the manifest schema documentation that could mislead an
installer into unsafe behavior), please open a GitHub issue or contact the
maintainers listed on the [ellmos-ai](https://github.com/ellmos-ai) organization
profile.
